Contoh Paragraf di HTML

Saat ini kita sedang berlatih dasar HTML pada mata kuliah Pemrograman Web. Materi awal adalah membuat tampilan web sederhana menggunakan tag dasar HTML.

Paragraf adalah kumpulan kalimat yang saling berhubungan sehingga membentuk ide pokok. Untuk membuat paragraf, digunakan elemen <p>.

Format Tulisan

Contoh penggunaan teks tebal, teks miring, teks dicoret, dan teks disisipkan.

Baris Baru vs Paragraf

Tag <br> hanya membuat baris baru.
Contoh baris kedua menggunakan <br>.
